# Incident Report Expo App

This app reports factory learning events and incidents to the existing backend.

## Flow

- Home
- Select report type
- Dynamic report form
- Success screen

## Backend API

- `POST /api/incidents`
- `GET /api/incidents`
- `GET /api/incidents/:id`

## Run

1. Install dependencies with `npm install`
2. Update `src/config/api.js` for your device/emulator
3. Start with `npm run start`
